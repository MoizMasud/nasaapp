About

A web application for displaying images from NASA, using Mongo, Express, Node.js and Angular 4. A rock solid Node.js/Express server using Mongoose to organize models and query the database, Angular-CLI to generate components, services and more. Local dev server and easy compilation, Full featured authentication using JSON web tokens. Login and store user data
